# Your First Bot

## Installation

You need to have following program installed:
- Node JS v6+
- NPM Package Management

To install, do:
```
$ npm install -g kata-cli
```

Login to kata-cli using:
```
$ kata login
```

## Creating bot

To create a bot you can type:
```
$ kata init <botid> <botname>
```

## Pushing bot

You can push your bot to the cloud:
```
$ kata push
```
